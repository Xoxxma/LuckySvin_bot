# LuckySvin_DD_bot

## UA

**LuckySvin_DD_bot** - це навчальний проєкт, створений у межах університетської практики.  
Проєкт реалізовано у форматі Telegram-бота з елементами текстової RPG / D&D-пригоди.  
Основна мета роботи - практичне застосування мови **C#**, платформи **.NET 10** та бібліотеки **Telegram.Bot** для створення інтерактивного застосунку з обробкою команд, повідомлень користувача та базовими ігровими механіками.

### Мета
Проєкт створено для закріплення практичних навичок розробки програмного забезпечення, роботи з Telegram Bot API, побудови логіки взаємодії з користувачем і реалізації простої ігрової системи.

### Можливості
- запуск Telegram-бота на C#
- обробка команди `/start`
- взаємодія з користувачем через текстові повідомлення
- елементи ігрової логіки
- механіка випадковості через кидки дайсів
- основа для подальшого розвитку проєкту

### Технології
- C#
- .NET 10
- Telegram.Bot

## ENG

**LuckySvin_DD_bot** is an educational project created as part of university practice.  
The project is implemented as a Telegram bot with elements of a text-based RPG / D&D-style adventure.  
Its main purpose is to apply **C#**, **.NET 10**, and **Telegram.Bot** in practice while building an interactive application with command handling, user interaction, and basic game mechanics.

### Purpose
The project was created to strengthen practical software development skills, work with the Telegram Bot API, design user interaction logic, and implement a simple game system.

### Features
- Telegram bot built with C#
- `/start` command handling
- user interaction through text messages
- game logic elements
- randomness mechanics through dice rolls
- foundation for future project expansion

### Tech Stack
- C#
- .NET 10
- Telegram.Bot
